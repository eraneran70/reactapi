# API

## Usage

All of the core components are exported as CommonJS modules. If you're using ES6 syntax, you can [import](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/import) the components you want to use from `hbc-core-components` like this:

```js
import { Alert } from 'hbc-core-components';
```
